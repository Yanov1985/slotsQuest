# ⚠️ ВАЖНО: ОСТАНОВИТЕ BACKEND ПЕРЕД ВЫПОЛНЕНИЕМ

## Шаги для восстановления:

1. **Найдите терминал с запущенным backend**
2. **Нажмите Ctrl+C** чтобы остановить сервер
3. **Подождите 2-3 секунды**
4. **Выполните команду:**

   ```bash
   node reset-and-restore-complete.js
   ```

5. **После завершения запустите backend:**
   ```bash
   npm run start:dev
   ```

## Ожидаемый результат:

- Backend запустится на порту 3001
- API будет доступно
- Данные отобразятся в админке и на frontend
